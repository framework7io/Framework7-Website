标签页(Tabs)

标签用于控制不同内容区块间的切换显示

Tabs标签布局

下方是标签组件的内容区HTML结构

<!-- Tabs 容器, 需要带有"tabs" class -->
<div class="tabs">
  <!-- 第一个tab, 需要带有"tab" class 和唯一的id -->
  <!-- 第一个tab是激活显示状态，通过添加"active" class实现 -->
  <div class="tab active" id="tab1">
    ... Tab 1 内容 ...
  </div>
  <!-- 第二个tab, 需要带有"tab" class 和唯一的id -->
  <div class="tab" id="tab2">
    ... Tab 2 内容
  </div>
  <!-- 第三个tab, 需要带有"tab" class 和唯一的id -->
  <div class="tab" id="tab3">
    ... Tab 3 内容 ...
  </div>
</div>

几点说明:

  • div class="tabs" - 所有tab外层共享一个容器. 没有他标签组件将无法工作
  • div class="tab" - 单独的tab，需要有 唯一的 id
  • div class="tab active" - 当前激活的 tab，即当前显示的tab，需要"active" class

各标签间切换

完成标签组件HTML布局后，我们需要一些控制面板以便用户可以在各标签间切换。

需要做的就是创建一个<a>，拥有名为"tab-link"的class，并且href 属性值和之前对应tab的id相同：

<!-- 关联到第一个标签, 需要设置和第一个标签id名相同的href属性(#tab1) -->
<a href="#tab1" class="tab-link active">Tab 1</a>
<!-- 关联到第二个标签, 需要设置和第二个标签id名相同的href属性(#tab1) -->
<a href="#tab2" class="tab-link">Tab 2</a>
<!-- 关联到第三个标签, 需要设置和第三个标签id名相同的href属性(#tab1) -->
<a href="#tab3" class="tab-link">Tab 3</a>

如你所见，第一个标签也有 "active" class。不是必需的，但如果所有这些a元素在相同的DOM树层级，内部脚本也会将“active”class添加到处于激活态的tab上。当你的“active”元素有不同的视觉样式时（比如Buttons Row / Segmented Control or links in Tab bar），这会很有用。

切换多重标签

上文描述的标记通过ID指定需要切换去的标签。但有时一个视图里会出现多块内容区响应同一个标签面板，这时我们可以用class代替id。标签面板里的a元素使用“data-tab”属性指向需要切换显示内容区的class（一般对应多块div）.比如 :

<!-- 顶部 Tabs -->
<div class="tabs tabs-top">
  <div class="tab tab1 active">...</div>
  <div class="tab tab2">...</div>
  <div class="tab tab3">...</div>
</div>
<!-- 底部 Tabs -->
<div class="tabs tabs-bottom">
  <div class="tab tab1 active">...</div>
  <div class="tab tab2">...</div>
  <div class="tab tab3">...</div>
</div>
<!-- Tabs 控制面板 -->
<div class="tab-links">
  <!-- 将同时切换标签内容区到 .tab1 区块-->
  <a href="#" class="tab-link" data-tab=".tab1">Tab 1</a>
  <!-- 将同时切换标签内容区到 .tab2 区块-->
  <a href="#" class="tab-link" data-tab=".tab2">Tab 2</a>
  <!-- 将同时切换标签内容区到 .tab3 区块-->
  <a href="#" class="tab-link" data-tab=".tab3">Tab 3</a>
</div>

注意data-tab 属性是包括tab/tabs的CSS选择器的字符串

动画过渡的标签

我们也可以使用transition动画过渡tab切换效果，只需要额外的class div class="tabs-animated-wrap"包裹在 div class="tabs"父级:

<div class="navbar">
  <div class="navbar-inner">
    <div class="center">
      <!-- Buttons row as tabs controller in navbar-->
      <div class="buttons-row">
        <!-- Link to 1st tab, active -->
        <a href="#tab1" class="tab-link active button">Tab 1</a>
        <!-- Link to 2nd tab -->
        <a href="#tab2" class="tab-link button">Tab 2</a>
        <!-- Link to 3rd tab -->
        <a href="#tab3" class="tab-link button">Tab 3</a>
      </div>
    </div>
  </div>
</div>
<div class="pages">
  <div class="page">
    <div class="page-content">
      <!-- Tabs animated wrapper, required to switch tabs with transition -->
      <div class="tabs-animated-wrap">
        <!-- Tabs, tabs wrapper -->
        <div class="tabs">
          <!-- Tab 1, active by default -->
          <div id="tab1" class="tab active">
            ... Tab 1 content ...
          </div>
          <!-- Tab 2 -->
          <div id="tab2" class="tab">
            ... Tab 2 content ...
          </div>
          <!-- Tab 3 -->
          <div id="tab3" class="tab">
            ... Tab 3 content ...
          </div>
        </div>
      </div>
    </div>
  </div>
</div>

注意最外层容器div.tabs-animated-wrap需要一个指定的高度，默认是其父亲节点100%的高度

标签的JS事件

当你需要在标签切换到active后做一些事情时，标签事件是很有用的:

Event Target Description
show 标签内容区块<div class="tab"> 该事件在目标内容区块被切换到可见状态时触发 

var myApp = new Framework7();
var $$ = Dom7;
$$('#tab1').on('show', function () {
    myApp.alert('Tab 1 is visible');
});
$$('#tab2').on('show', function () {
    myApp.alert('Tab 2 is visible');
});
$$('#tab3').on('show', function () {
    myApp.alert('Tab 3 is visible');
});

使用JS切换标签

也可以通过JS切换标签，我们需要调用以下的应用方法:

myApp.showTab(tab)

  • tab - HTMLElement or string 标签标记（完整CSS选择器。必需
  • 当标签切换成功后方法会返回 true,否则会返回false
<a href="#" class="show-tab-1">Show Tab 1</a>
<a href="#" class="show-tab-2">Show Tab 2</a>
<a href="#" class="show-tab-3">Show Tab 3</a>
var myApp = new Framework7();
var $$ = Dom7;
$$('.show-tab-1').on('click', function () {
    myApp.showTab('#tab1');
});
$$('.show-tab-2').on('click', function () {
    myApp.showTab('#tab2');
});
$$('.show-tab-3').on('click', function () {
    myApp.showTab('#tab3');
});
