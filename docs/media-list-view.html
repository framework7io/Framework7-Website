媒体列表视图（Media List View）

媒体列表视图是列表视图（list view）的拓展，它是为了展示更复杂的数据比如产品，服务，用户等等.

当然，它的布局更为复杂:

<!-- Media list block has additional "media-list" class -->
<div class="list-block media-list">
    <ul>
        <li>
            <div class="item-content">
                <div class="item-media">
                    <img src="path/to/img.jpg">
                </div>
                <div class="item-inner">
                    <div class="item-title-row">
                        <div class="item-title">Element title</div>
                        <div class="item-after">Element label</div>
                    </div>
                    <div class="item-subtitle">Subtitle</div>
                    <div class="item-text">Additional description text</div>
                </div>
            </div>
        </li>
    </ul>
</div>

上面的代码:

  • item-media 和 item-inner的主要弹性包裹元素（flex wrapper）. 必选元素.

    • item-media - 媒体元素（如图标，图片等）的容器. 可选元素.

    • item-inner - item-title 和 item-after的主要弹性包裹元素（flex wrapper）. 必选元素.

      • item-title-row - item-title 和 item-after的主要弹性包裹元素（flex wrapper）. 可选元素.

        • item-title - 单行列表项目标题. 可选元素.

        • item-after - 列表项目标签. 可以包含其他额外的html标签（如label text, or badge, or switch/toggle or button等）. 可选元素.

      • item-subtitle - 额外的单行标题. 可选元素.

      • item-text - 额外的，两行布局的，细节描述区域的容器. 可选元素.
