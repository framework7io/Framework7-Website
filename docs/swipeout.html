滑动操作列表Swipeout (Swipeable List)

可滑动列表(Swipeout)是列表视图list view的拓展，它额外提供滑动操作的功能，滑动列表元素可以展现隐藏的功能菜单，如滑动删除这样的操作.

让我们来看一下滑动列表元素的布局结构:

<div class="list-block">
  <ul>
    <!-- Additional "swipeout" class on li -->
    <li class="swipeout">
      <!-- Usual list element wrapped with "swipeout-content" -->
      <div class="swipeout-content">
        <!-- Your list element here -->
        <div class="item-content">
          <div class="item-media">...</div>
          <div class="item-inner">...</div>
        </div>
      </div>
      <!-- Swipeout actions -->
      <div class="swipeout-actions">
        <div class="swipeout-actions-inner">
          <!-- Swipeout actions links/buttons -->
          <a href="#">Action 1</a>
          <a href="#">Action 2</a>
        </div>
      </div>
    </li>
    ...
  </ul>
</div>

上面的代码:

  • swipeout-content - 列表里的元素的包裹元素. 这个容器会随着滑动操作移动
  • swipeout-actions - 当你有滑动操作菜单（swipeout action buttons/links）时，需要将操作菜单包含在这个包裹元素中
  • swipeout-actions-inner - 滑动操作菜单的容器，这个容器会在滑动的过程中显示

注意swipeout-content和swipeout-actions应该是<li>的直接子节点

如果只有"item-content"样式类的子节点，可以将"item-content"样式类加到"swipeout-content"元素上，以获取布局，简化结构:

<li class="swipeout">
  <div class="swipeout-content item-content">
    <div class="item-media">...</div>
    <div class="item-inner">...</div>
  </div>
  <div class="swipeout-actions">
    <div class="swipeout-actions-inner">
      <a href="#">Action 1</a>
      <a href="#">Action 2</a>
    </div>
  </div>
</li>

如果使用的是链接元素（link elements），你可以将上述的样式类都加到链接元素上:

<li class="swipeout">
  <a href="#" class="swipeout-content item-content item-link">
    <div class="item-media">...</div>
    <div class="item-inner">...</div>
  </a>
  <div class="swipeout-actions">
    <div class="swipeout-actions-inner">
      <a href="#">Action 1</a>
      <a href="#">Action 2</a>
    </div>
  </div>
</li>

滑动删除（Swipe To Delete）

在Framework7中使用滑动删除不需要写Javascript代码，只需要将<a class="swipeout-delete">链接加入到滑动菜单区中:

<li class="swipeout">
  <div class="swipeout-content item-content">
    <div class="item-media">...</div>
    <div class="item-inner">...</div>
  </div>
  <div class="swipeout-actions">
    <div class="swipeout-actions-inner">
      <!-- Add this button and item will be deleted automatically -->
      <a href="#" class="swipeout-delete">Delete</a>
    </div>
  </div>
</li>

可以在点击删除按钮的时候调用确认询问模态窗口（Confirm modal），只有在确认后才会将需要删除的元素从列表中移除. 使用确认功能只需要把"data-confirm"和"data-confirm-title" (可选)属性加到删除菜单链接（<a class="swipeout-delete">）上:

<li class="swipeout">
  <div class="swipeout-content item-content">
    <div class="item-media">...</div>
    <div class="item-inner">...</div>
  </div>
  <div class="swipeout-actions">
    <div class="swipeout-actions-inner">
      <!-- We add data-confirm and data-confirm-title attributes -->
      <a href="#" class="swipeout-delete" data-confirm="Are you sure want to delete this item?" data-confirm-title="Delete?">Delete</a>
    </div>
  </div>
</li>

滑动列表（Swipeout） JavaScript API

滑动操作列表提供丰富的JavaScript API来控制滑动列表元素。下面是提供的属性及方法:

myApp.swipeoutOpen(el) - 在指定的元素中显示滑动菜单

  • el - HTMLElement or string (with CSS Selector) of list (<li>) element with "swipeout" class. 必选的

myApp.swipeoutClose(el) - 在指定的元素中隐藏滑动菜单

  • el - HTMLElement or string (with CSS Selector) of list (<li>) element with "swipeout" class. 必选的

myApp.swipeoutDelete(el) - 删除指定的滑动列表中的元素

  • el - HTMLElement or string (with CSS Selector) of list (<li>) element with "swipeout" class. 必选的

myApp.swipeoutOpenedEl - 属性（property）. 已打开的滑动列表元素；当没有滑动列表中的元素被打开，返回undefined（Contains link to currently opened swipeout HTMLElement. Or equal to 'undefined' if none of elements are currently opened）

滑动列表事件（Swipeout events）

滑动列表元素会触发以下事件:

事件（Event） 目标（Target） 说明（Description）
swipeout Swipeout Element<li class="swipeout"> 当滑动列表元素被移动时触发，event.detail.progress表示当前正在打开的菜单的开启进度（Event will be triggered while you move swipeout element. event.detail.progress contains current opened progress percentage）
open Swipeout Element<li class="swipeout"> 当滑动列表元素开始显示时触发（Event will be triggered when swipeout element starts its opening animation）
opened Swipeout Element<li class="swipeout"> 当滑动列表元素完成显示时触发（Event will be triggered after swipeout element completes its opening animation）
close Swipeout Element<li class="swipeout"> 当滑动列表元素开始隐藏时触发（Event will be triggered when swipeout element starts its closing animation）
closed Swipeout Element<li class="swipeout"> 当滑动列表元素完成隐藏时触发（Event will be triggered after swipeout element completes its closing animation）
delete Swipeout Element<li class="swipeout"> 当滑动列表元素开始删除操作时触发（Event will be triggered after swipeout element starts its delete animation）
deleted Swipeout Element<li class="swipeout"> 当滑动列表元素被删除后触发（Event will be triggered after swipeout element completes its delete animation right before it will be removed from DOM） 
var myApp = new Framework7();
 
var $$ = Dom7;
 
$$('.swipeout').on('deleted', function () {
  myApp.alert('Item removed');
});
 
$$('.swipeout').on('swipeout', function (e) {
  console.log('Item opened on: ' + e.detail.progress + '%');
});
